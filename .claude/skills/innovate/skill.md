# Innovate

Manage innovation ideas from capture to execution.

## Usage

- `/innovate` — show all ideas and projects
- `/innovate add "<idea>"` — evaluate a new idea
- `/innovate work <project>` — work on an existing project

## Add Mode

1. User describes the idea
2. Evaluate together:
   - What problem does it solve?
   - How hard is it to build? (weekend / month / quarter)
   - Who would use it?
   - What makes it unique?
3. Score and file as `idea-YYMMDD-description.md` in the right topic folder
4. If promising, suggest promoting to `project-*/` folder

## Dashboard

Scan `KNOWLEDGE/*/idea-*` and `KNOWLEDGE/*/project-*`. Show:
- Ideas by topic
- Active projects (have recent activity)
- Dormant projects (no activity in 30+ days)
