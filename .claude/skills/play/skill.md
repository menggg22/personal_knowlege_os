# Play

Playful exploration. Nights and weekends. No pressure.

## Usage

- `/play` — show exploration options
- `/play random` — pick something random
- `/play random mashup` — connect two unrelated topics

## Execution

1. Scan `KNOWLEDGE/` for all topics, ideas, learnings, projects
2. Present options by mood:
   - 🧠 Deep thinking (papers, architecture, complex topics)
   - 🛠️ Hands-on (build something, prototype, experiment)
   - ✍️ Creative (blog topics, writing ideas)
   - 🎨 Pure curiosity (anything interesting)
3. Wait for user choice, then explore freely

### Random Mashup

1. Pick two unrelated topics from `KNOWLEDGE/`
2. Find an unexpected connection
3. Explore where it leads

## During Exploration

- Follow tangents — this is play, not work
- If something good emerges, suggest capturing it:
  - Add to `FUN/sparks.md` (keep loose)
  - Create a proper entry in `KNOWLEDGE/` (if substantial)
- No todos, no pressure, no guilt

## After

If sparks emerged, write a light session note to `FUN/sessions/YYYY-MM-DD-topic.md`.
